# DDFD-FaceRec Web 演示

基于训练好的模型提供完整的 Web 界面测试人脸识别功能。

## 🚀 快速启动

### Windows

双击运行：
```
start.bat
```

或命令行运行：
```bash
cd web
..\venv\Scripts\python app.py
```

### Linux/Mac

```bash
cd web
python app.py
```

## 📱 访问地址

启动后访问：**http://localhost:7860**

## ✨ 功能模块

### 1. 📷 人脸检测
- 上传图像自动检测人脸
- 显示人脸位置和置信度
- 支持多人脸检测

### 2. 🔍 特征提取
- 提取 512 维人脸特征向量
- 显示特征范数和部分特征值
- 支持指定人脸索引

### 3. 👥 1:1 人脸验证
- 比对两张图像是否为同一人
- 显示相似度和判定结果
- 阈值可配置（默认 0.6）

### 4. 🔎 1:N 人脸搜索
- 注册人脸到数据库
- 在库中搜索相似人脸
- 返回 Top-K 匹配结果

### 5. 📋 人脸库管理
- 查看已注册人脸
- 清空数据库
- 统计信息展示

## 📊 模型信息

| 模块 | 模型 | 参数量 | 推理速度 |
|-----|------|--------|---------|
| 检测 | DKGA-Det | 64.82M | ~2.4s |
| 识别 | DDFD-Rec | 27.99M | ~6ms |
| 比对 | HNSW | - | 0.4ms@1000 |

## 🔧 配置选项

### 环境变量

```bash
# 设置设备（cuda/cpu）
export DEVICE=cuda

# 设置服务端口
export GRADIO_SERVER_PORT=7860

# 设置检测阈值
export DETECT_SCORE_THRESH=0.5

# 设置验证阈值
export VERIFY_THRESHOLD=0.6
```

## 📝 使用示例

### 注册人脸

1. 进入"1:N 人脸搜索"标签页
2. 上传包含人脸的图像
3. 输入姓名或 ID
4. 点击"注册人脸"

### 搜索人脸

1. 上传查询图像
2. 选择人脸索引（如果有多张脸）
3. 设置返回数量（Top-K）
4. 点击"开始搜索"

### 1:1 验证

1. 进入"1:1 人脸验证"标签页
2. 分别上传两张图像
3. 选择各自的人脸索引
4. 点击"开始验证"
5. 查看相似度和结果

## ⚠️ 注意事项

1. **首次加载**：首次启动需要加载模型，可能需要 10-30 秒
2. **显存占用**：约 2-4GB（RTX 4090）
3. **并发限制**：建议单用户测试使用
4. **图像格式**：支持 JPG、PNG 格式
5. **人脸大小**：建议人脸尺寸 > 50x50 像素

## 🐛 故障排除

### 问题：启动失败

**解决方案**：
```bash
# 检查依赖
pip install -r requirements.txt

# 检查端口占用
netstat -ano | findstr 7860

# 更换端口
python app.py --port 7861
```

### 问题：检测不到人脸

**解决方案**：
- 确保图像清晰，光线充足
- 人脸正对摄像头
- 人脸尺寸足够大（>50x50）
- 调整检测阈值：`score_thresh=0.3`

### 问题：识别准确率低

**解决方案**：
- 使用更清晰的图像
- 确保注册和查询图像角度相似
- 增加训练数据重新训练模型
